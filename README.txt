"# CSN-Miniproject-V1M-5" 

alarmcentrale code, dit wordt een alarmcentrale die, een code vereist. 
Dit alarm kent 3 fasen, 
1- het alarm staat uit er is een juiste code ingevoerd 
2- er is een sensor afgegaan je hebt X seconden om je code in te voeren 
3- je hebt een onjuiste code ingevoerd of je hebt er te lang over gedaan er wordt een email notification naar de bewoner gestuurd.
